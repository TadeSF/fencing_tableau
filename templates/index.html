FenceWithFriends helps you create and manage the fencing tournament of your dreams.

Easily organize and manage FENCING TOURNAMENTS

FenceWithFriends makes it easy to manage big and small fencing tournaments with different rulesets, including (multiple) preliminary groups, repechage and placement matches. It tracks scores and calculates standings and results in real-time. Save time and focus on what matters most - the competition.

It is free and open-source. There is no registration required.

You just need a Startlist as a comma delimited CSV file. You can download a template and sample file and read the Instructions on how to format the Startlist CSV below.

The Startlist File

The startlist file is a comma delimited CSV file. The first line contains the column names. The following columns (with one header line) are required:

  • Name: The name of the fencer
  • Club: The club of the fencer â€“ to avoid display annomalies, please just use an acronym (e.g. NOT "Turngemeinde in Berlin 1848 e.V.", but "TIB")
  • Nationality: The Nationality of the fencer. Must be a three-letter alpha-3 country code (e.g. "GER" for Germany, "FRA" for France, "USA" for the United States of America, etc.)
  • Gender (optional): The gender of the fencer. Must be either
    • "M"
    • "F"
    • "D" for divers
  • Handedness (optional): The handedness of the fencer. Must be either
    • "R"
    • "L"
  • Age (optional): The age of the fencer. Must be a positiv integer, either
    • (preferably) the age (e.g. 23)
    • the birthyear (e.g. 1999, age calculations might be one year off due to missing birthday information)
    • the birthday (format: YYYY-MM-DD, e.g. 1999-01-01)

How it Works

FenceWithFriends is a Python-based application that hosts a webserver in a local (wifi) network. This way, it is possible for everyone to see current standings and matches from their mobile devices. It just needs one person as a tournament manager with a computer.
